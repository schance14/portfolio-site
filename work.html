MeetUp App!

Meet-up is a serverless progressive web application that gives its users a way to search for upcoming events by city to meet people and find interesting experiences.

Meet-up is built with React and uses the test driven development technique to deliver high quality code. MeetUp App uses serverless functions with AWS Lambda for its authorization server and Google Calendar API to fetch events and OAuth2 for online authorization.

FilmFinder is a single-page web appplication for users to browse movies, read up on information on specific movies, and create their own "favorites" list. FilmFinder is a responive Javacript application that works in conjunction with a REST API and MongoDB Atlas database.
